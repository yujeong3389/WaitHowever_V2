# Wait, However
#### 가짜뉴스를 정확하게 판별할 수 있을까?'라는 문제의식에서 프로젝트를 시작했습니다. 
#### 하지만 가짜뉴스를 100% 정확하게 판별하는 것은 현실적으로 어렵다고 판단했습니다. 
#### 이에 저희는 기사의 진위를 단정하기보다, 기사의 정치 성향과 선동성, 사실과 의견을 시각화하여 사용자가 스스로 객관적으로 판단할 수 있도록 돕는 방향으로 서비스를 설계했습니다. 
#### Gemini API를 활용해 기사의 정치 성향, 선동성, 사실과 의견을 분석하고, Chrome 확장 프로그램을 통해 사용자가 기사를 읽는 화면에서 바로 분석 결과를 확인할 수 있도록 구현했습니다. 
#### 이를 통해 사용자가 뉴스를 더욱 비판적으로 읽을 수 있는 환경을 제공합니다.
